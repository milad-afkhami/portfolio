### _gives us a better API to deal with react portal._

### Source

<Code
  language="javascript"
  code={`import { createPortal } from "react-dom";
  
export function isomorphicPortal(children, selector) {
  const isClient = !!(typeof window !== "undefined" && window.document && window.document.createElement);
  if (!isClient) return null;
  
  const destEl = selector ? document.querySelector(selector) : null;
  return createPortal(children, destEl || document.body);
}
`}
/>

### Example

<Code
  language="javascript"
  code={`isomorphicPortal(<Div />, "#modal__wrapper");`}
/>
