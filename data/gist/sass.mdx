I've created a nice solution to generate CSS utility classes dynamically using [Sass At-rules](https://Sass-lang.com/documentation/at-rules).

## 1. Color Utility Classes

### 1.1. Generating Color Utility Classes

```css
/* .bg-[transparent, brand, primary, secondary, success, error, warning, info] */
@each $key, $value in $bg-colors {
  .bg-#{$key} {
    background-color: $value !important;
  }
}

/* .text-[primary, secondary, tertiary] */
@each $key, $value in $text-colors {
  .text-#{$key} {
    color: $value !important;
  }
}

/* .border-[light, dark] */
@each $key, $value in $border-colors {
  .border-#{$key} {
    border: 1px solid $value !important;
  }
}
```

### 1.2. Variables For Generating Color Utility Classes

<details>

<summary>
  <strong>Toggle</strong>
</summary>

```css
$text-colors: (
  primary: /* PRIMARY TEXT COLOR VALUE */,
  secondary: /* SECONDARY TEXT COLOR VALUE */,
  tertiary: /* TERTIARY TEXT COLOR VALUE */
);
$bg-colors: (
  transparent: transparent,
  brand: /* BRAND COLOR VALUE */,
  primary: /* PRIMARY BG COLOR VALUE */,
  secondary: /* SECONDARY BG COLOR VALUE */,
  tertiary: /* TERTIARY BG COLOR VALUE */,
  success: /* ASSISTIVE SUCCESS COLOR VALUE */,
  error: /* ASSISTIVE ERROR COLOR VALUE */,
  warning: /* ASSISTIVE WARNING COLOR VALUE */,
  info: /* ASSISTIVE INFO COLOR VALUE */
);
$border-colors: (
  light: /* LIGHT BORDER COLOR VALUE */,
  dark: /* DARK BORDER COLOR VALUE */
);
```

</details>

## 2. Font Utility Classes

### 2.1. Generating Font Utility Classes

```css
/* .font-[xxxlg, xxlg, xlg, lg, m, sm, xsm, xxsm] */
@each $size, $value in $font-sizes {
  .font-#{$size} {
    font-size: 1rem + $value !important;
  }
}
```

### 2.2. Variables For Generating Font Utility Classes

<details>

<summary>
  <strong>Toggle</strong>
</summary>

```css
$font-sizes: (
  xxxlg: 1rem,
  xxlg: 0.5rem,
  xlg: 0.25rem,
  lg: 0.125rem,
  md: 0rem,
  sm: -0.125rem,
  xsm: -0.25rem,
  xxsm: -0.5rem
);
```

</details>

## 3. Flex Utility Classes

### 3.1. Generating Flex Utility Classes

```css
/* .flex-[center, end, start]-[center, end, start, around, between] e.g. "flex-center-between" */
@each $align, $align-value in $flex-align-values {
  .flex-#{$align} {
    display: flex;
    align-items: $align-value;

    @each $justify, $justify-value in $flex-justify-values {
      &-#{$justify} {
        display: flex;
        align-items: $align-value;
        justify-content: $justify-value;
      }
    }
  }
}
```

### 3.2. Variables For Generating Flex Utility Classes

<details>

<summary>
  <strong>Toggle</strong>
</summary>

```css
$flex-align-values: (
  center: center,
  end: flex-end,
  start: flex-start
);
$flex-justify-values: (
  center: center,
  end: flex-end,
  start: flex-start,
  around: space-around,
  between: space-between
);
```

</details>

@each $key, $value in $curves {
